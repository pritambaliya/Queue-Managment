# Queue Management App

A simple Queue Management application built using React that allows users to add people to a service queue by selecting their name and required service. The app demonstrates core React concepts such as state management, props, form handling, and component-based architecture.

## Features
- Add users to a queue with name and service type
- Form validation to prevent empty submissions
- Clear input fields after successful submission
- Clean and responsive UI
- Data flow from child to parent component using props

## Built With
- React.js
- JavaScript (ES6)
- Tailwind CSS
- Vite
